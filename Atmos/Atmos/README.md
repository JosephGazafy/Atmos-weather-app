# Atmos

A professional-grade atmospheric simulation tool for Termux.

## Features
- **Modular Architecture**: Clean separation of core physics and CLI logic.
- **Optimized for Termux**: Built to leverage OpenBLAS and LAPACK on Android.
- **Verified**: Fully unit-tested formulas.
